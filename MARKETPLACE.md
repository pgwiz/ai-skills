Description

Agent Memory provides global and per-project memory files, safe editing rules, and one-command install for GitHub Copilot and other .agents/skills runtimes. It helps agents avoid destructive operations and persist context across sessions.

Key features

- Cross-platform installers (PowerShell, bash)
- Per-project `.agent/` memory files (MEMORY.md, CODEBASE.md, CURRENT_TASK.md)
- Global protocol files for consistent behavior across projects
- README.env generation to document environment variables
